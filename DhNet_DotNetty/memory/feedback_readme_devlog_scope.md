---
name: README/DEVLOG 범위 제한
description: README와 DEVLOG에 코드 리뷰 내용을 포함하지 않는다
type: feedback
---

코드 리뷰 결과는 README나 DEVLOG에 포함하지 않는다. dev/ 디렉터리의 별도 문서에만 기록한다.

**Why:** 사용자가 코드 리뷰 내용을 README/DEVLOG에 추가했을 때 명시적으로 제거를 요청했다.

**How to apply:** 코드 리뷰, 개선 과제, 버그 목록 등은 dev/ 하위 문서에만 작성한다. README는 프로젝트 소개/실행법, DEVLOG는 개발 이력(구현한 것)에만 집중한다.
